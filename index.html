HTML

HTML: Stands for "HyperText Markup Language"

  • HTML provides the "framework" or "structure" for an application
  • The "head" element contains information about the webpage
  • The "body" element represents the visible content shown to the user
  • The "p" element is used to write text ie (think paragraph)
  • The "ul" element is used to make bulleted lists
  • The "li" element is used to indicate items in a list

CSS

CSS: Stand for "Cascading Style Sheets"

  • CSS provides "style" to an application
  • There are three ways to apply CSS: inline CSS, internal CSS, and external CSS
      • Inline CSS: Can be used to change to style of a single element
      Internal CSS: By placing the "style" element in the "head" element
      External CSS: Allows for seperation of your CSS code making it more managable for larger projects
    (It is important to note that when doing so you must link your HTML and CSS together)
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds changed files in working directory to the staging area of git
  • git commit: commits changes to repository
  • git commit -m "message-name": commits changes to repository with a message indicator

JavaScript

  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
  • Variables:
      • String: This is a sequence of text known as a string (To signify that the value is a string, enclose it in single or double quote marks)
      Boolean: This is a True/False value(The words true and false are special keywords that don't need quote marks)
      Array: This is a structure that allows you to store multiple values in a single reference
      Object: This can be anything, everything in JavaScript is an object and can be stored in a variable
  • /* "insert text" */: Ommits text between it
